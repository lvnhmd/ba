What is Fashion Ave UI Kit

Fashion Ave UI Kit is a number of useful user interface components built on Bootstrap 3.1.0 framework. We did not modify any line of Bootstrap, so you can safely use Fashion Ave UI Kit in your ongoing project.

What is included

Once purchased, unzip the compressed folder to see the directory structure of Fashion Ave UI Kit. You'll see something like this:

Note: You can modify css files directly, but it's better to modify less files and generate css from them.

Browser support

Since Fashion Ave UI Kit is built on Bootstrap 3 framework, it supports the same browsers as Bootstrap.

Getting Started

To start working with Fashion Ave UI Kit you can use a basic template.html, or take the completely functional pages included in this package. Since it is built on Bootstrap, you can also use it on your project with Bootstrap 3.

We provide you LESS version for better customization of your project. We built it Bootstrap way so you won't find big difference. Use less/variables.less to modify different styles and style.less to remove needless components. After all modifications you will need to recompile style.less and new style.css file will be generated with new variables. For compiling less files you can use WinLess but you can use less compiler you like.

After finishing the customization part you'll have to drop all contents of html folder to your project folder and including style.css file in your HTML page: <link href="style.css" media="screen" rel="stylesheet">

If you want to start from scratch, the best way is to use our template where all startup files are already included and ready to be used. You can find template.html file in /html/ folder.

    jQuery(document).ready(function() {
        var graphwidth = $('.widget-graph .inner').width();
        $('.widget-graph .graph').css('height', parseInt(graphwidth/1.6));
        $(window).resize(function() {
            graphwidth = $('.widget-graph .inner').width();

        var d1 = [[0.5, 7], [1.3, 15], [2, 11], [3, 20], [3.8, 14], [4.2, 17], [4.6, 16], [5.8, 32]],
                graphholder = $("#graph"),
                plot = $.plot(graphholder, [d1], {
                    colors: ["#ffffff", "#94cfdf"],
                    xaxis: {
                        min: 0,
                        max: 6
                    },
                    yaxis: {
                        min: 0,
                        max: 40
                    },
                    series: {
                        lines: {
                            show: true,
                            lineWidth: 4,
                            fill: true,
                            fillColor: "rgba(255, 255, 255, 0.2)"
                        },
                        points: {
                            show: true,

                        borderColor: "#81c8e5"
                    }

        function showTooltip(x, y, contents) {
            $("<div id='graph-tooltip'>" + contents + "</div>").css({top: y - 90, left: x - 43}).appendTo("body").fadeIn(200);
        }

        var previousPoint = null;
        graphholder.on("plothover", function (event, pos, item) {

            if (item) {
                if (previousPoint != item.dataIndex) {

                    previousPoint = item.dataIndex;

                    $("#graph-tooltip").remove();

                            y = item.datapoint[1].toFixed(2);

                    showTooltip(item.pageX, item.pageY, y*10 + '<span>new clothes</span>');
                }
            } else {
                $("#graph-tooltip").remove();
                previousPoint = null;
            }
        });
    });
